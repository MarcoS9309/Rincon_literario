# Rincón Literario

**Por M. Vinicio**

Un sitio web estático dedicado a la literatura, la poesía y la meditación espiritual.

## 📚 Contenido

### Historias (7)
Relatos narrativos personales y ficticios que exploran temas universales de la experiencia humana.

### Cuentos (15)
Cuentos de ciencia ficción y fantasía que transportan al lector a mundos imaginarios.

### Poemas (10)
Colección de poemas líricos que expresan emociones y reflexiones sobre la vida.

### Meditaciones (3)
Guías espirituales para el rezo del Santo Rosario y meditaciones religiosas.

### Novena y Meditación Personal (9)
Programa de meditación diaria estructurada sobre los Siete Dolores de María.

## 🌐 Navegación

- **Página principal**: `index.html` - Portal de navegación con todas las obras organizadas por categorías
- **Contenido estático**: Carpeta `static/` con 44 páginas HTML individuales
- **Archivos fuente**: Contenido original en formato Markdown

## 🛠️ Características Técnicas

- ✅ **Sitio web completamente estático** - Sin dependencias de JavaScript
- ✅ **Diseño responsivo** - Compatible con dispositivos móviles y de escritorio
- ✅ **44 páginas HTML generadas** - Una por cada obra literaria
- ✅ **Navegación intuitiva** - Organizado por categorías temáticas
- ✅ **Diseño moderno** - Gradientes CSS y tipografía profesional

## 📁 Estructura del Proyecto

```
Rincon_literario/
├── index.html              # Página principal de navegación
├── style.css               # Estilos CSS globales
├── database.json           # Índice de todas las obras
├── static/                 # Páginas HTML individuales (44 archivos)
├── Historias/              # Archivos fuente de historias (.md)
├── cuentos/                # Archivos fuente de cuentos (.md)
├── poemas/                 # Archivos fuente de poemas (.md)
├── santo_rosario/          # Archivos fuente de meditaciones (.md)
├── novena_dolores/         # Archivos fuente de novena (.md)
└── generate_improved.py    # Script generador de páginas estáticas
```

## 🚀 Uso

1. **Navegación local**: Abrir `index.html` en cualquier navegador web
2. **Servidor local**: Ejecutar `python3 -m http.server 8000` para servir en `http://localhost:8000`
3. **Hosting web**: Subir todos los archivos a cualquier servidor web estático

## ✨ Características Destacadas

- **Sin JavaScript**: Funciona en cualquier navegador, incluso con JavaScript deshabilitado
- **Acceso offline**: Funciona completamente sin conexión a internet
- **Carga rápida**: Archivos optimizados para carga instantánea
- **Contenido preservado**: Formato original mantenido con correcciones técnicas
